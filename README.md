# 公交司机排班管理系统

## 项目简介

全栈公交司机排班管理系统，支持管理员创建排班计划、冲突检测、司机签到签退、考勤统计等功能。采用日历视图直观展示排班信息。

## 技术栈

- **前端**：Vite + React 18 + TypeScript（默认端口 5174）
- **后端**：Express + TypeScript + better-sqlite3（默认端口 3202）
- **认证**：JWT + bcryptjs
- **数据库**：SQLite（better-sqlite3）
- **启动方式**：concurrently 同时启动前后端

## 启动方式

```bash
# 安装依赖（根目录 + server + client）
npm run install:all

# 启动开发环境（前后端同时启动）
npm run dev
```

启动后访问 http://localhost:5174

## 功能说明

### 管理员功能
- 排班日历视图：按月查看所有排班，日历格子显示当班司机
- 创建排班：选择日期范围、线路，将司机分配到早班/中班/晚班
- 冲突检测：同一司机同一天不能排两个班次
- 考勤统计后台：查看每条线路出车率、司机出勤率、异常统计
- 按月导出排班表和考勤报表（CSV 格式）

### 司机功能
- 个人排班日历：查看自己的排班安排
- 班次详情：点击日期查看线路、时段、车牌号
- 签到打卡：上班前签到
- 签退打卡：下班签退
- 迟到/早退自动标记异常

## 注意事项

- 首次启动自动建表并 seed 预置线路和司机数据
- 管理员密码 admin/123456，司机测试账号 driver/123456
- SQLite 数据库文件位于 server/data.db
- 前端通过 Vite proxy 将 /api 请求转发到后端 3202 端口
- 可通过 `VITE_PORT`、`VITE_API_PORT`、`PORT` 覆盖默认端口
